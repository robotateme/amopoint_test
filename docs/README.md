# Документация

- [deploy.md](deploy.md) — Docker runtime, Fly.io config и ручной deploy workflow.
- [operations.md](operations.md) — Redis cache, quality gates, команды `make`, сборка и проверка.
- [rfc/0001-architecture-and-quality-gates.md](rfc/0001-architecture-and-quality-gates.md) — архитектурное RFC.
